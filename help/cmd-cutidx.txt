cutidx subcommand
    Print specified column, record offset and length.

    Usage:
        $ xsvutils --jvm cutidx [<COLUMN>] [option...]

    Parameter:
        <COLUMN>          (optional) Print <COLUMN> (same as --col)

    Options:
        --col <COLUMN>     Print <COLUMN>
        --help             Print help information about this subcommand.

    Either of COLUMN parameter or --col option is required.

    To see `xsvutils help filter` for examples.
